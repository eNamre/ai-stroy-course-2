# Промпт-пак: сметчик — ЖК «Уютный дом на Матвеева»

*Один из 6 ролевых паков. Формат идентичен паку ПТО (принцип OpenAI Academy Prompt Packs: роль + контекст + задача + формат под конкретный рабочий сценарий). Готов к копированию в GigaChat / YandexGPT / ChatGPT. Данные — из пакета «Уютного дома» (файлы 01–23). `[...]` — подставьте своё.*

## Как пользоваться
- **Обязательно приложите документ, указанный в промпте (📎 файл №…).** Без него модель начнёт выдумывать факты.
- **Грунтовка против выдумок (добавляйте к любому промпту):** «Используй только данные из приложенного документа. Если данных не хватает — не выдумывай, а перечисли, чего именно недостаёт. Факты, которых нет в документе (ФИО, реквизиты, нормы), помечай как `[уточнить]`.»
- Скопируйте промпт, замените `[...]`, приложите нужный документ кейса.
- **Красные линии:** не загружайте ПДн (ФЗ-152) и коммерческую тайну (файл 20); **итоговые цифры смет и суммы к оплате — только в профильном ПО (ГРАНД-Смета и т.п.)**, ИИ — для понимания, черновика и проверки сходимости, не для утверждённого расчёта.
- **Приём «шаг назад»:** в сложных задачах сначала спросите принцип/критерии, потом — применение (примеры ниже помечены 🔁).
