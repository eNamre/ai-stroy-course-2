# Промпт-пак: инженер ПТО — ЖК «Уютный дом на Матвеева»

*Образец ролевого промпт-пака (один из 6 по строительным ролям). Формат — по принципу OpenAI Academy Prompt Packs: каждый промпт под конкретный рабочий сценарий, со структурой роль + контекст + задача + формат. Готов к копированию в GigaChat / YandexGPT / ChatGPT. Данные — из пакета «Уютного дома» (файлы 01–23). `[...]` — подставьте своё.*

## Как пользоваться
- **Обязательно приложите документ, указанный в промпте (📎 файл №…).** Без него модель начнёт выдумывать факты.
- **Грунтовка против выдумок (добавляйте к любому промпту):** «Используй только данные из приложенного документа. Если данных не хватает — не выдумывай, а перечисли, чего именно недостаёт. Факты, которых нет в документе (ФИО, реквизиты, нормы), помечай как `[уточнить]`.»
- Скопируйте промпт, замените `[...]`, приложите нужный документ кейса.
- **Красные линии:** не загружайте ПДн (ФЗ-152) и коммерческую тайну (см. файл 20); итоговые цифры смет — только в профильном ПО; юридически значимые решения и подписи — за человеком.
- **Приём «шаг назад»:** в сложных задачах сначала спросите принцип/критерии, потом — применение (примеры ниже помечены 🔁).
