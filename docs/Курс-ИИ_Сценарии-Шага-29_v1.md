# Сценарии Шага 29 — «Мини-финал: неделя роли с ИИ» (5 ролей)

*Спека для будущей сборки 5 HTML-финалов по образцу `Курс-ИИ_Урок-Мини-финал-ПТО_v1.html` (драматургия: интро «как устроен мини-финал» → 5 эпизодов пн–пт → живой чат → «мой ИИ-набор роли» → чек-лист самопроверки). Кейс — ЖК «Уютный дом на Матвеева». Каждый эпизод: раскрывающийся документ кейса (`details.doc`), приём/квиз/выбор варианта, демо-ответ ИИ, ссылка на промпт из ролевого пака. Роли и их паки: ПТО (готов, Шаг 10/образец), **стройконтроль, сметчик, прораб, ГИП, ОТ** — сценарии ниже. Промпт-паки: `Курс-ИИ_Промпт-пак-Стройконтроль_v1.md`, `-Сметчик_v1.md`, `-Прораб_v1.md`, `-ГИП_v1.md`, `-ОТ_v1.md`.*

## Как устроена спека каждого сценария

- **Сквозная нить** — какой сюжет разворачивается через неделю (из паспортов шагов и логики кейса).
- **Таблица 5 эпизодов** пн–пт: день → документ кейса (№ файла) → задача с ИИ → промпт из пака роли (код) → результат/развязка.
- **Интерактивы** (2–3) — какой тип виджета мини-финала ставим на этот эпизод (по образцу ПТО: выбор варианта с фидбеком, квиз, раскрывающийся документ + демо-ответ ИИ, «найди расхождение»).
- **Файлы кейса** — какие документы и медиа-сырьё прикладывать при сборке HTML.
