# Курс «ИИ в строительстве» — 30 шагов (v2)

Рабочая версия курса для встраивания в платформу через iframe (GitHub Pages).
Продолжение прототипа [ai-stroy-course](https://github.com/eNamre/ai-stroy-course) — тот репозиторий заморожен как есть, вся новая работа здесь.

## Правило стабильных URL
Каждый урок живёт под постоянным именем без номера версии (iframe-ссылки в платформе не должны ломаться). История правок — в коммитах.

## Позиционирование
Это **демо-комплект**: команда Актион переносит материалы в свою платформу. В каждом уроке — блок «Материалы урока» (архив в `materials/`), аннотации «⚙ Для переноса» показывают, где платформа подключает свои механики (куратор, прогресс, роли). Мир примеров — единый кейс «ЖК Уютный дом на Матвеева».

## Состав
- `index.html` — программа курса (индекс v5, «30 шагов»)
- `karta-kursa.html` — карта курса
- `module-0.html` — Шаг 0, онбординг (Модуль-0 v2)
- `lesson-1.html` — Шаг 5, «ИИ не знает вашу компанию» (Урок-1 v3)
- `lesson-kp.html` — Шаг 16, сравнение смет и КП (Урок-КП v7, эталон)
- `lesson-ozhr.html` — Шаг 18, голос → ОЖР (Урок-ОЖР v1)
- `shablon-uroka.html` — шаблон урока для авторов (блоки + виджет «ИИ-ассистента»)

Источник правды по составу шагов: `Курс-ИИ_Паспорта-шагов_v1.md` (рабочая папка проекта).

- `materials/shag-XX.zip` — архивы материалов по урокам (тексты, документы кейса, интерактив)
